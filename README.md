# 🚀 Featured Projects

## ⚡ LinearFlow

**Smart Campus Workflow & Event Management System**

A digital governance platform designed to automate campus operations.

Key capabilities:

• Secure admin-verified onboarding
• Role-based dashboards
• Clash-free event scheduling
• Multi-level approval workflow
• Digital attendance system
• Automated institutional reporting

---

## 🎓 Transparent-Exam

**AI-Powered Answer Sheet Evaluation System**

An intelligent evaluation platform that analyzes answer sheets using AI.

Core technologies:

- • OCR-based text extraction
- • Semantic answer evaluation
- • Confidence-based scoring
- • AI feedback generation

---

## 🛡 AI Phishing Website Detection

**Cybersecurity Platform for Detecting Malicious Websites**

A machine-learning based security system designed to detect phishing websites in real time.

Capabilities include:

• URL feature analysis
• ML-based phishing detection
• Browser extension monitoring
• Security warning alerts

---

## 🏛 CivicPulse

**Civic Issue Reporting Platform**

A civic-tech solution that allows citizens to report public infrastructure problems and enables authorities to track resolution progress.

---

## 🌱 EcoMind

**AI-Powered Sustainability Learning Platform**

Encourages eco-friendly behavior through:

• Gamified sustainability challenges
• AI learning assistant
• Community environmental engagement
